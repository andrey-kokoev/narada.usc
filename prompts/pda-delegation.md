# PDA Delegation Prompt

Use this before handing work to an executor.

```text
Apply Progressive De-Arbitrarization.
Do not implement the presenting task while hidden decision-relevant arbitrariness remains.
First localize the governing ambiguity.
Separate forced structure from contingent policy.
Preserve already-closed concrete decisions.
Keep authority loci, state transitions, and failure semantics explicit.
When residual ambiguity becomes explicit and enumerable, ask one numbered question at a time with lettered options and best default marked (*).
Recompose only when the remaining ambiguity is forced, explicitly chosen, assigned as residual, or irrelevant to execution.
Then produce an implementation-ready task with transformation, evidence requirement, and review predicate.
```
