# AI Agent Operation Question Map

## Principal Authority

- What is the agent's charter? What is it allowed and not allowed to do?
- What tools or APIs can the agent invoke?
- What decisions require human approval before execution?
- What is the review posture (pre-approval, post-review, no review)?
- What safety constraints apply?

## Semantic Authority

- How does the agent access knowledge and context?
- What are the escalation rules?
- What evaluation criteria define correct performance?
