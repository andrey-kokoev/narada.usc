# Principal Intent

## Principal

Alice Chen, maintainer of `widget-lib`.

## Statement

> "widget-lib has a public API that developers struggle to use because there is no documentation site. I want a public documentation site that stays synchronized with the code."

## Success Criteria

- A new developer can learn how to use the three main widgets (`Button`, `Card`, `Modal`) from the site alone.
- The site updates automatically when the library API changes.
- The site is hosted publicly and linked from the repo README.

## Constraints

- Must use open-source or free-tier tooling.
- Must not require maintainers to manually copy-paste code examples.
- Must not break existing CI.

## Authority

Alice holds principal authority. She may delegate execution and review, but final say on scope and success criteria remains hers.
